## Notes for Implementer
- Cost-sensitive: prefer Copilot first, escalate to Claude only if stuck
- Health bars: top corners, one per player. Show corgi portrait placeholder (colored rectangle) + heart pips or bar.
- Special meter: below health bar. Glow effect when full (simple color change or alpha pulse).
- Combo counter: center-top, text showing "x2", "x3", etc. Fades out ~1s after combo breaks.
- Boss banner: big banner at top with boss name text and health bar. FlashPhaseChange = brief white flash on the bar.
- Pause: TogglePause sets Time.timeScale to 0/1. Input action map switches to UI map during pause per tech doc.
- Safe area: use Screen.safeArea to get the safe rect, apply as padding to the root Canvas RectTransform. iOS notch and home indicator must not clip UI elements.
- No UI polish — just functional elements. Visual polish is not this issue's job.
